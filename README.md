# Roost — AI Property Management Platform

Manage short-term rental properties with AI-powered automation.

## Quick Start

1. Clone & Install: `git clone https://github.com/mbis1/Roost.git && cd Roost && npm install`
2. Set up Supabase: Create project, run `supabase-schema.sql` in SQL Editor
3. Configure: `cp .env.example .env.local` and fill in Supabase credentials
4. Run: `npm run dev` then open http://localhost:3000
5. Deploy: Push to GitHub, import in Vercel, add env vars

## Tech Stack

Next.js 14, React, Tailwind CSS, Supabase (Postgres + Auth + RLS), HuggingFace AI (free), iCal sync, Vercel hosting
